## About The Project

This project is aimed to built a dynamic pokemon webpage using Angular framework and PokeAPI. It showcases three main pages; login, pokemon list and profile.

The login is an area where user can create and login with just a username.
The pokemon list is an area where user can catch pokemons and release pokemons.
The profile is an area where user can see the caught pokemons and has the option to release the pokemons.

See [React_Lost in Translation.pdf](https://github.com/AhmadKhodabaks/pokepedia-ng/blob/master/Angular_Pokemon Trainer.pdf) for more detailed requirements for this assignment.

Created by Ahmad Khodabaks & Meg Stefouli.

## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/AhmadKhodabaks/pokepedia-ng.git
   ```
2. Run the following command in your CLI to install the required dependecies on your localhost.
   ```sh
   npm install
   ```
3. It is important that you set your API key and url in the environment files. You can use Noroff Assignment API.
   ```sh
   https://github.com/dewald-els/noroff-assignment-api
   apiUsers: "",
   apiKey: "",
   apiPokemon: "",
   apiSprites: ""
   ```   
4. Run the following command in your CLI to run on localhost.
   ```sh
   ng serve
   ```
It will run the app in the development mode. Open http://localhost:4200 to view it in your browser.
